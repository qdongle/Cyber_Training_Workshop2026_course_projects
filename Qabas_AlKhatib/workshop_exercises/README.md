## Workshop Exercises

- `03_pyscf_prism/` — Episode 3: PySCF/Prism electronic structure exercises
  (CIS/TDHF, TDA-TDDFT, SR-ADC, EOM-CCSD, SA-CASSCF, NEVPT2, SOC-NEVPT2, CVS-IP-MR-ADC)

  (geometry/Hessian, CASSCF/CASPT2 optimization, oscillator strengths, spin-orbit coupling,
  photoelectron spectrum, MECI, nonadiabatic coupling, core-hole XPS)
- `05_pyspawn/` — Episode 5: Ab initio multiple spawning exercises
