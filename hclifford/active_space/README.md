This folder contains all of the files relevant to obtaining an active space with SA-CASSCF for TBHP in OpenMolcas

tbhp_sa4_casscf_sp.input is the input file. It is the single point calculation of TBHP at SA(4)-CASSCF(6e,4o)/cc-pVDZ level of theory

tbhp_sa4_casscf_sp.log is the output file 

tbhp_sa4_casscf_sp.rasscf.molden is the molden file containing the final optimized SA-CASSCF orbitals

The tbhp_unopt.Oxyz is the initial unoptimized geometry of TBHP
