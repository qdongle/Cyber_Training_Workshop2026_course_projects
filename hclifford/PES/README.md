This folder contains all of the files relevant to calculating the relaxed scan PES of TBHP across the RO-OH bond stretching with SA-CASSCF for TBHP in OpenMolcas

There are individual directories at certain RO-OH bond lengths

relaxed_scan_tbhp_casscf.input is the input file. It is the single point calculation of TBHP at SA(4)-CASSCF(6e,4o)/cc-pVDZ level of theory

molcas.stdout is the output file
