# TDDFT Excitation Analysis of Small Molecular Systems Using PySCF

CyberTraining 2026 Project

Author: Saif Ullah
Research Area: Electronic Structure Theory / Spin Qubits / Porous Materials / Low Dimensional Materials

## Project Overview

This project investigates vertical electronic excitation energies of small
molecules using time-dependent density functional theory (TDDFT) calculations
performed with the PySCF quantum chemistry package.

The systems studied are:

- Diazomethane (CH2N2)
- Cyanoacetylene (HCCCN)
- Methyl isocyanate (CH3NCO)

The goal is to analyze excitation energies, oscillator strengths, orbital
transitions, and basis-set convergence behavior.

---

## Computational Approach

Electronic excitation calculations were performed using:

- TDDFT
- CAM-B3LYP exchange-correlation functional
- Dunning correlation-consistent basis sets:
  - cc-pVDZ
  - cc-pVTZ
  - cc-pVQZ

Excitation energies were evaluated and compared across basis sets.
